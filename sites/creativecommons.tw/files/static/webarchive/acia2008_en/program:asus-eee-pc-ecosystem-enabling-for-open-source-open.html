Program » Asus EeePc Ecosystem Enabling - For Open Source & Open Content

Abstract

Eee PC is a breakthrough in design methodology and a revolution in user experience. We call this ''Eeexperience."

The Eee PC desktop comes from the open-sourced Linux system, and is now planning to integrate open content with “Commons” ideas. Besides that, Asus is also pleased to announce the upcoming release of the Eee PC SDK . This will allow open source and open content communities to bring software and content to the Eee PC in ways that are easy to develop, easy to port, and easy to release.
