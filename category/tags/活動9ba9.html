再會了

感謝您對「台灣創用CC計畫」的支持與愛護，十多年來「台灣創用CC計畫」受中央研究院支持，並在資訊科學研究所以及資訊科技創新研究中心執行，現已完成階段性的任務。CC 授權條款在台灣的推廣使用，之後將由「開放文化基金會」協調組織的「CC台灣社群 」進行。

本網站預計持續維運至 2021年底，網站內容基本上不會再更動。

Ubuntu Free Culture Showcase

2008-08-08

想讓作品揚名國際的機會又來了一個。Ubuntu 為下一個版本(應該是 8.10，也就是今年十月會推出的版本)舉辦了「Free Culture Showcase」活動，一旦入選、你的作品就可能會「灑」到全球數百萬台電腦上 — 這比台灣現在很多藝人賣得都多啦！以下引自 Ubuntu Wiki，並附上簡譯：

For a long time now, Ubuntu has included on the CD a small collection of media files that can be used to show off a new Ubuntu system. This content has always included a sound and a video clip, both using the free Ogg Vorbis and Theora formats.

「第二屆兩岸交流映象獎」鼓勵以創用CC授權媒材創作

2008-06-06

由行政院陸委會主辦的「2008兩岸交流映象獎」鼓勵各大專院校的學生使用創用CC授權釋出的媒材進行創作，並於即日起受理報名至10月10日止。

本競賽內容以兩岸關係為主的創意短片為主，呈現方式可以短劇、動畫或是宣傳片，長度不可以超過兩分鐘。作品必須是自行創作或是從創用CC全球分享網站，依照授權方式下載使用。獲獎作品將有獎金。相關的活動查詢可至陸委會查詢。

開放文化相關研討會再添一成員：Open Tech Summit Taiwan

4月25日到 4月29日，開放原始碼組織 Open Hardware Initiative 臺灣社群與華碩電腦合作協辦第一屆的「Open Tech Summit Taiwan」，同時邀請OSSF 的林誠夏、Wikia 的 KJ 及創用 CC 一同參加。

Open Tech Summit Taiwan 企圖營造社群通力合作的感覺，是以邀集了各式開放文化的相關組織與個人一同參加。...

「博物館創用CC推廣說明會」開放報名中

2008-04-22

博物館這類館藏單位，典藏著大量的數位化著作，如何透過創用CC授權條款，以保留部分著作權利的方式，加速著作的流通與再利用，釋出公眾創意加值，激發整體社會的創意能量，成為博物館領域不可不正視的重要課題。

由行政院國家科學委員會、行政院文化建設委員會、數位典藏與數位學習國家型科技計畫辦理的「博物館創用CC推廣說明會」，將於97年4月30日（三）上午9：00在南港台灣創意設計中心舉行。邀集對博物館授權事務有興趣的各界人士共襄盛舉！相關詳細內容，請見報名網頁。

iSummit 2008 開放壽司食譜投稿

2008-04-15

別再找藉口不參與創作及分享了，現在連你阿媽都可以投 iSummit 的稿！

iSummit '08 將在七月底於日本札榥舉行，主辦單位特別結合當地文化、辦了個「CC 世界盃壽司創作大賽」。即日起至 5 月 16 日止，你可以將你創作的壽司食譜寄到主辦單位處，優勝者的食譜將以創用 CC 授權，並刊登在札榥市政府的網站上喔！(是也有獎品的樣子，但沒有寫得很明確 ^^; )

詳細的比賽細節請參考 iSummit 08 Sushi Project 網站。

8/1-8/2，到劍潭混搭你的創意：OLPC Curriculum Jam！

2007-07-30

台灣「創用CC」計畫與 OLPC (One Laptop Per Child; 每童一機) 合作在台北推動這次的 OLPC Curriculum Jam 活動，本活動由維基媒體國際大會在正式會議前所主辦，時間為8月1日至8月2日。OLPC Jam 是一種內容創作的新體驗，讓所有的參加者，在短短的幾天當中將想像化為現實。針對這次的 Jam，我們將會一同打造學習活動、製作小型素材，讓小朋友得以從中自我學習，教學內容不論是數學、藝術或是民俗舞蹈皆可。在第二天下午，由本地學童所組成的評審團，將會檢視這些作品，並且宣佈獲勝團隊。
